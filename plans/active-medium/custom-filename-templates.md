# Plan: Custom Filename Templates

## Status
Pending

## Priority
Medium

## Summary
Allow users to customize output filename patterns. Currently filenames are hardcoded to `YYYY-MM-DD_HH-MM-SS_%3f.mp4` format. A template system enables better file management and integration with external tools.

## Current State
- Output filenames generated by `generate_output_filename()` with hardcoded format
- No user customization of naming patterns
- Cannot include game name, resolution, encoder, or custom text in filenames
- Content creators need organized, searchable filenames

## Implementation Steps

### 1. Template Variables
- Define supported template variables:
  - `{date}` — YYYY-MM-DD
  - `{time}` — HH-MM-SS
  - `{datetime}` — YYYY-MM-DD_HH-MM-SS
  - `{game}` — Detected game name (or "Desktop")
  - `{resolution}` — e.g., "1080p", "1440p"
  - `{fps}` — Framerate
  - `{encoder}` — Encoder name (e.g., "amf", "nvenc")
  - `{counter}` — Auto-incrementing number to avoid collisions
  - `{custom}` — User-defined text prefix/suffix

### 2. Template Engine
- Simple string interpolation (no complex logic needed)
- Validate template at config load time
- Provide default template: `{datetime}_{counter}.mp4`
- Sanitize game names and custom text for filesystem safety

### 3. Configuration
- Add `filename_template: String` to `GeneralConfig`
- Validate template contains at least one variable
- Auto-append `.mp4` extension if not present

### 4. GUI
- Add template input field to the General settings tab
- Show available variables as a reference list
- Live preview of the generated filename
- Validation error display for invalid templates

### 5. Backward Compatibility
- Default template matches current hardcoded format
- Existing clips are unaffected

## Files to Modify
- `crates/liteclip-core/src/config/config_mod/types.rs` — Add `filename_template` field
- `crates/liteclip-core/src/output/functions.rs` — Template interpolation in `generate_output_filename`
- `src/gui/settings.rs` — Template input field and preview in General tab

## Estimated Effort
Small (1-2 days)

## Dependencies
- None

## Risks
- Template variables with special characters (game names with slashes) need sanitization
- Counter logic must handle collisions across sessions
- Very long templates could exceed filesystem path limits
